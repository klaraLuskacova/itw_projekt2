# Profilové portfolio – Klára Luskačová

## Popis projektu
Tento projekt je osobní prezentace vytvořená v rámci předmětu ITW.  
Cílem bylo vytvořit přehlednou webovou stránku, která prezentuje základní informace o mně, mém vzdělání, projektech a dovednostech.

## Funkcionalita
- navigace mezi sekcemi pomocí menu
- zvýraznění aktivní sekce při scrollování
- vizuální efekty při hover
- responzivní design (mobil)
- hamburger menu pro mobil
- tlačítko pro návrat na začátek stránky

## Obsah stránky
- Úvod
- O mně
- Vzdělání
- Projekty
- Dovednosti
- Kontakt

## Použití AI
Při tvorbě projektu byly využita AI jako pomoc při řešení některých složitějších částí, zejména při implementaci JavaScriptu a návrhu dekorativních prvků - tečky na okrajích stránky a gradient čáry u nadpisů.
