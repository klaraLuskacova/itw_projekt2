Projekty

Školní a technické projekty

V rámci studia jsem pracovala na několika projektech, které mi pomohly lépe pochopit programování, návrh aplikací a práci s větším zadáním. Každý projekt byl zaměřený na jinou oblast a vyžadoval jiný způsob řešení.

01

Interpret jazyka SOL26

Projekt spočíval v implementaci interpretu jednoduchého objektově orientovaného jazyka SOL26. Interpret přijímá program v podobě XML reprezentace abstraktního syntaktického stromu a provádí jej včetně práce s třídami, dědičností, metodami, objekty a běhovým prostředím.

Python XML OOP Docker
1 týden IPP, 2. ročník

Co jsem se naučila

  • Návrh interpretu a běhového prostředí
  • Práce s XML a validace vstupních dat
  • Kontejnerizace pomocí Dockeru
  • Statická analýza kódu nástrojem Ruff
02

Databázová aplikace

Týmový projekt zaměřený na návrh a implementaci relační databáze v Oracle 12c. Součástí bylo vytvoření ER diagramu, SQL skriptů pro tvorbu schématu, dotazů SELECT, triggerů, uložených procedur a materializovaných pohledů.

SQL Oracle 12c PL/SQL ER diagram
4 dny Týmový projekt IDS, 2. ročník

Co jsem se naučila

  • Návrh relačního databázového schématu
  • Psaní komplexních SQL dotazů
  • Databázové triggery a uložené procedury
  • Optimalizace dotazů pomocí indexů a EXPLAIN PLAN
03

Síťový L4 scanner

Implementace nástroje pro skenování TCP a UDP portů zadaného zařízení na síti. Program odesílá pakety přes sockety, analyzuje odpovědi a vyhodnocuje stav portů jako open, closed nebo filtered. Součástí projektu bylo také vytvoření testů.

C++ TCP/IP UDP libpcap Sockety
1 týden IPK, 2. ročník

Co jsem se naučila

  • Práce se síťovými sockety v C++
  • Principy TCP a UDP komunikace na nízké úrovni
  • Zachytávání síťového provozu pomocí libpcap
  • Zpracování argumentů příkazové řádky
